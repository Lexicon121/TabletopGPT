# Knowledge Data

Place local, lawfully usable knowledge files here. This repository intentionally includes no copyrighted tabletop PDFs, private campaign notes, or character sheets.

- `uploads/`: original user-provided files, ignored by git.
- `processed/`: extracted/chunked text, ignored by git.
- `indexes/`: local indexing artifacts, ignored by git.

RAG is intended for rules lookup, lore lookup, character sheet reference, house rules, item descriptions, NPC notes, and world knowledge. It must not directly mutate game state.
